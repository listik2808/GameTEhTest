Использовать Unity 6000.3.2f1 !!! https://unity.com/ru/releases/editor/archive

Есть игра, сделана под гугл. Надо собрать сборку под яндекс игры, самое главное скорость загрузки и размер файла. Игра чтобы смотрелось приятно, не уродство типа пережата.

На что обратить внимание:
1) Элементы игры не должны обрезаться/выходить за рамки экрана. Обрезается игровое поле.
2) На Android устройствах плеер игры отображается в панели уведомлений.
3) Проверьте, что язык игры управляется через Yandex Games SDK
4) Чтобы элементо нормально вставлялись, я про кусочки динозавров.
5) Проверьте на телефоне и у себя на компьютере
6) Плюсом будет порядок в папках
7) Большим плюсом знание addressables unity!
